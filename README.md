## 项目运行

```
git clone git@github.com:cemcoe/xbook.git

cd xbook

npm install

npm run dev
```

## 项目目录结构

- assets 静态资源 css imgs
- views 路由组件
- components 公共组件
- utils 功能函数
- store pinia 代码
- service 网络请求相关

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
