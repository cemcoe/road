## 项目运行

```
git clone git@github.com:cemcoe/xbook.git

cd xbook

npm install

npm run dev
```

## 项目目录结构

- assets 静态资源 css imgs
- views 路由组件
- components 公共组件
- utils 功能函数
- store pinia 代码
- network 网络请求相关

更多信息可参考 [开发流水账](https://www.jianshu.com/nb/47900649)
